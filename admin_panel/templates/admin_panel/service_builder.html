How eligibility gates work An eligibility gate checks whether the user's answer meets a condition. If it fails, the applicant is shown the fail message and blocked from proceeding. Choose an operator, enter the threshold, and write a clear fail message.
Example: type = number  ·  operator = gte  ·  value = 18  →  "Must be at least 18 years old."
